# TEACHERS HUB
Teachers Hub is a simple Python-based project designed to help teachers manage basic academic tasks efficiently. 
It provides an easy-to-use platform that supports organization and improves daily teaching activities. 
The project demonstrates the practical use of Python in creating useful and user-friendly educational applications.

---

# FEATURES
1. Stores questions database.
2. Creates testpapers with desired questions.
3. Has shortcut options for notepad, calculator and Google Chrome.

---

# TECH STACK
1. PYTHON
2. Libraries (random, time, subprocess, tkinter)
